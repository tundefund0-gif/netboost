# Netboost

Local DNS cache for Android — no root required.

## Quick start

1. Download APK from [Actions](https://github.com/tundefund0-gif/netboost/actions)
2. Install on Android 8.0+ (turn **Private DNS** Off)
3. Open app → tap **Start**
4. Notification shows hit/miss counters

## Source

- **android/** — VpnService app (single-file service ~170 lines)
- `dns/`, `main.go` — Go CLI prototype (archived)
